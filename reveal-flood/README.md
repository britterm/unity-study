# Unity Flood Reveal Deck

Reveal.js presentation for the Unity men's Bible study session on the flood.

## Run Locally

```bash
npm install
npm run dev -- --port 5173
```

Open:

```text
http://127.0.0.1:5173/
```

## Build

```bash
npm run build
```

The production build is written to `dist/`.

## Navigation

- Horizontal arrows move between major sections.
- Vertical arrows move within a section.
- Press `s` for speaker view.

## Source Notes

- Full Scripture reading slides use KJV, public domain.
- Research source trails remain in `../flood/*.md`.
- Embedded images were extracted from the exported Keynote/PowerPoint deck.
