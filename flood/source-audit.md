# Source Audit

Status: initial research pass complete. All sources below still need final quote-level verification before slide production.

## Verification Checklist

For each quote or source note:

- Link works.
- Author and work are identified.
- Chapter, section, page, or paragraph is included when available.
- Quote is short and exact.
- Translation is named when known.
- Source type is identified: primary, near-primary, advocate, critic, or secondary.
- The note does not overclaim what the source says.

## Items To Verify Before Presentation

- Exact wording and translation for all church father quotes.
- Whether historical sources explicitly argue for flood extent or merely assume it.
- Whether modern source summaries fairly represent the advocate's own argument.
- Whether any scientific claims are necessary for the study goal.

## Self-Review Notes

- Avoid overclaiming patristic evidence. Many early sources assume universality but do not debate modern global/local geography.
- Use BibleGateway mainly for reference links, not long quotation.
- Prefer exact primary-source quotations only after checking the original page and translation.
- Several sources are advocate ministry sites. That is acceptable for steel-manning, but mark them as advocates.
- Some subagent suggestions need checking before presentation:
  - The exact Irenaeus quote "the deluge came upon the whole earth" should be verified in context before use.
  - The exact Tertullian phrase "the ark was a figure of the Church" varies by translation; the Souter translation says the Church "was represented by the ark."
  - Chrysostom, Bede, Theodoret, and Origen need better primary access or print translation before quoting.
  - Gavin Ortlund source is a podcast aggregation link, not ideal for primary citation; use only after finding the original Truth Unites page/video.
  - Wikipedia links are included only as orientation for hard-to-access sources, not as final authority.
